\# AutoParts.kz — Интернет-магазин автозапчастей



\## О проекте

Магазин автозапчастей в стиле Flip.kz. Визуально и по удобству — как Flip, но под авто-тематику.



\## Технологии

\- Next.js 14

\- Tailwind CSS

\- Node.js (портативный, D:\\nodejs)

\- GitHub: https://github.com/Lex793/auto-parts-shop



\## Структура проекта



\### Страницы (src/app/)



| Страница | Файл | Описание | Статус |

|----------|------|----------|--------|

| Главная | `page.js` | Поиск, подбор по авто/VIN, категории, товары | ✅ Готово, адаптивно |

| Каталог | `catalog/page.js` | Фильтры слева, сортировка, сетка товаров | ✅ Готово, адаптивно |

| Товар | `product/page.js` | Фото, характеристики, таблица совместимости, "С этим покупают" | ✅ Готово, адаптивно |

| Корзина | `cart/page.js` | 3 шага: контакты → доставка → оплата, рекомендации | ✅ Готово, адаптивно |

| Профиль | `profile/page.js` | История заказов, избранное, адреса | ✅ Готово, адаптивно |



\### Компоненты (src/components/)

\- `Search.js` — поиск с автодополнением (если создан)
