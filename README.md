# AutoParts.kz — Интернет-магазин автозапчастей

[![Next.js](https://img.shields.io/badge/Next.js-14-black?logo=next.js)](https://nextjs.org)
[![React](https://img.shields.io/badge/React-18-blue?logo=react)](https://react.dev)
[![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)](https://www.typescriptlang.org)
[![Tailwind](https://img.shields.io/badge/Tailwind-3-cyan?logo=tailwind-css)](https://tailwindcss.com)

Современный интернет-магазин автозапчастей на **Next.js 14 (App Router)** с полной воронкой продаж, админ-панелью и подбором по автомобилю.

---

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск разработки
npm run dev

# Сборка для продакшена
npm run build
npm run start
```

**Открыть:** http://localhost:3000

---

## 📦 Функционал

### Страницы (15+)

| Страница | URL | Описание |
|----------|-----|----------|
| Главная | `/` | Баннеры, категории, популярные товары |
| Каталог | `/catalog` | Фильтры, сортировка, пагинация |
| Товар | `/product/[id]` | Фото, характеристики, совместимость, отзывы |
| Корзина | `/cart` | Управление товарами, рекомендации |
| Оформление | `/checkout` | 3 шага: контакты → доставка → оплата |
| Заказы | `/account/orders` | История заказов, повторный заказ |
| Личный кабинет | `/account` | Профиль, заказы, настройки |
| Избранное | `/account/favorites` | Сохранённые товары |
| Сравнение | `/compare` | Сравнение до 4 товаров |
| VIN-проверка | `/vin-check` | Подбор по VIN-коду (17 символов) |
| Подбор по авто | `/car-selector` | Марка → Модель → Год → Двигатель |
| Админка | `/admin` | CRUD товаров и заказов |
| Доставка | `/delivery` | Условия доставки и возврата |

### Ключевые особенности

✅ **Адаптивный дизайн** (mobile-first)  
✅ **Поиск по SKU/артикулу** (точный + частичный)  
✅ **VIN-поиск** с маской ввода (17 символов, uppercase)  
✅ **Подбор по автомобилю** (марка/модель/год/двигатель)  
✅ **Карточка товара** с фото, характеристиками, совместимостью  
✅ **Блок "С этим покупают"** (cross-sell)  
✅ **Аналоги товаров** (кросс-номера)  
✅ **Корзина** с рекомендациями фильтров  
✅ **3-шаговое оформление** заказа с валидацией  
✅ **Личный кабинет** с историей заказов  
✅ **Повторный заказ** в 1 клик  
✅ **Избранное и Сравнение** товаров  
✅ **Админ-панель** с CRUD товаров  
✅ **Загрузка фото** товаров (base64)  
✅ **Управление остатками** (stock)  
✅ **Дерево категорий** (родитель → подкатегории)  
✅ **SEO-оптимизация** (динамические meta-теги)  
✅ **Хлебные крошки** с полной иерархией
